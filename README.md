# Zomato Restaurant Success Analysis

## Project Objective
The objective of this project is to analyze Zomato restaurant data and identify the key factors that influence restaurant success, such as location, cuisine, and pricing.

## Dataset
- Source: Kaggle (Zomato Restaurants Dataset)
- Records: Approximately 9,500 entries
- Key Features: City, Cuisines, Average Cost for Two, Aggregate Rating, Votes

## Tools and Technologies
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib

## Analysis Performed
- Data cleaning and preprocessing
- Handling missing values
- City-wise restaurant distribution analysis
- Cuisine popularity analysis
- Price vs rating relationship
- Identification of top-performing cities and cuisines based on average rating

## Key Insights
- Delhi NCR has the highest concentration of restaurants.
- North Indian cuisine is the most dominant category.
- Mid-range restaurants tend to receive higher customer ratings.
- Higher pricing does not necessarily lead to better ratings.
- Cities with the highest number of restaurants are not always the highest rated.

## Visualizations
- Bar charts for city and cuisine analysis
- Scatter plot for price vs rating analysis
- Heatmap for correlation analysis

## Conclusion
This project demonstrates how exploratory data analysis can be used to derive meaningful business insights from real-world datasets and support data-driven decision making.
